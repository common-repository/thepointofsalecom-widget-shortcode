== Description ==
Thepointofsale.com allows you to add shopping carts and purchase windows in your articles and WordPress pages via Gutenberg blocks by activating the extension (<https://thepointofsale.com>)
This extension also supports the old way of embedding your short codes generated by our platform.

== Installation ==
1. Install thepointofsale.com either through the WordPress.org plugin directory or by downloading the files to your server.
2. Activate the extension via the extension page of your WordPress administration.
3. Use Gutenberg Blocks from thepointofsale.com to generate a shopping cart or purchase window in your articles and pages.
4. Sell tickets!
